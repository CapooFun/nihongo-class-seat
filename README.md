# nihongo-class-seat

日本语言学校班级座位表静态网页，适合直接部署到 GitHub Pages。

## 文件结构

- `index.html`：主页面
- `css/style.css`：样式
- `js/data.js`：老师、同学、座位与节假日数据
- `js/app.js`：日期判断、课程渲染、座位渲染、弹窗逻辑
- `DESIGN.md`：设计文档

## 当前实现

- 今日日期与上课/休息状态
- 周一到周五课程表
- 日本 2026 年法定节假日判断
- 5 x 6 教室座位表
- 点击老师或同学弹出详情
- 深色科技蓝移动端样式

## 后续你可以继续补的内容

- 每位同学的兴趣、JLPT、留言
- 更准确的座位位置
- 老师或同学头像图片
- GitHub 仓库创建与 Pages 部署
